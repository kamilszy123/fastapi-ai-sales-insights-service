# AI Sales Insights API

Backend service for AI-powered sales analysis based on Allegro CSV exports.

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- OpenAI API
- Docker
- Alembic
- JWT Authentication

## Project Goals

This project is focused on learning and demonstrating:

- modern backend architecture
- FastAPI and async Python
- AI API integrations
- Docker-based development workflow
- layered and testable backend design
- production-oriented backend practices

## Planned Features

- CSV upload and processing
- AI-generated sales insights
- trend and seasonality analysis
- JWT authentication
- background tasks
- provider abstraction for AI services
- structured JSON AI responses
- Dockerized environment
- automated tests

## Architecture

The project follows a layered backend architecture with clear separation of concerns:

- API layer
- services
- providers/integrations
- repositories
- database layer
- schemas and validation

## Status

Project currently in early development phase.
